#  Statistic Calculator

A Python-based Statistical Calculator built using Object-Oriented Programming (OOP). This project performs common statistical calculations on a dataset and helps users analyze numerical data efficiently.

##  Features

* Calculate Mean
* Calculate Median
* Calculate Mode
* User-Friendly Menu System
* Object-Oriented Design
* Handles Multiple Numerical Inputs

##  Technologies Used

* Python
* Object-Oriented Programming (OOP)

##  Project Structure

##  How to Run

### Clone the Repository

```bash
git clone https://github.com/Gautam2-468/Statistic-Calculator.git
```

### Navigate to the Project Folder

```bash
cd Statistic-Calculator
```

### Run the Program

```bash
python calculator.py
```
